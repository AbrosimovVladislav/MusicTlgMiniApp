# Project Idea — Music Expert App

## Суть

Маркетплейс платных консультаций для музыкальных (и смежных) креаторов.
Начинающие музыканты, продюсеры, авторы находят экспертов в своей нише и получают персональную консультацию за деньги.

---

## Проблема

Начинающие креаторы хотят получить обратную связь, совет или разбор от опытного человека в их нише — но нет удобного места, где это можно сделать быстро, понятно и безопасно.

---

## Решение

Telegram Mini App, где:
- **Клиент** формирует запрос, выбирает эксперта и платит за консультацию
- **Эксперт** ведёт профиль, получает входящие запросы и сам находит клиентов

---

## Пользователи

### Клиент (креатор)
Начинающий или развивающийся музыкант / продюсер / автор, которому нужна экспертная обратная связь.

**Сценарий:**
1. Регистрируется в приложении
2. Формирует запрос — описывает, что хочет обсудить
3. Выбирает категорию и подкатегорию своей ниши
4. Получает список подходящих экспертов
5. Выбирает эксперта, оплачивает, получает консультацию

### Эксперт
Опытный специалист в музыкальной нише, который монетизирует свою экспертизу.

**Сценарий:**
1. Регистрируется, заполняет профиль (ниша, опыт, стоимость, описание)
2. Попадает в список экспертов по своей категории/подкатегории
3. Получает входящие запросы от клиентов, которые выбрали его напрямую
4. Видит общую ленту запросов по своей нише — может сам откликнуться на интересные

---

## Ключевая механика

```
Клиент создаёт запрос
    → указывает категорию + подкатегорию
    → видит список экспертов по этой нише
    → выбирает эксперта → платит → консультируется

Эксперт
    → получает прямые запросы от клиентов
    → видит ленту всех запросов по своей нише
    → может сам откликнуться на запрос
```

## Категории (предварительно)

- **Музыкальное производство** (биты, сведение, мастеринг, саунд-дизайн)
- **Вокал** (постановка голоса, запись, стиль)
- **Авторство** (тексты, мелодика, аранжировки)
- **Карьера и продвижение** (питчинг, дистрибуция, бренд артиста, соцсети)
- **Теория и образование** (гармония, импровизация, инструменты)
